Bracket Pair Colorization and accessibility

Bracket pairing can be a pain to pick out. VS Code enables bracket pair colorization natively (previously used an extension).

Overall customization as below, but consider colorblind palettes as well.

// Up to six possibilities
"workbench.colorCustomizations": {
  "editorBracketHighlight.foreground1": "#E69F00",
  "editorBracketHighlight.foreground2": "#56B4E9",
  "editorBracketHighlight.foreground3": "#009E73",
  "editorBracketHighlight.unexpectedBracket.foreground": "#db6165"
}
