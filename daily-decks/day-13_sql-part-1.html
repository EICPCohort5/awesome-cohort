Summary

  • SQL commands:
    • SELECT
    • INSERT
    • UPDATE
    • DELETE
  • There are many ways to write select statements

Summary: Data modification

  • INSERT INTO [table] [fields] VALUES 'val1', number1, 'val2', ...
  • UPDATE [table] SET field = value, ... WHERE field = value
    • WHERE clause is optional, but recommended
  • DELETE FROM [table] WHERE field = value
    • WHERE clause is optional, but recommended

Summary: Select statements

  • SELECT 'static', 'values'
  • SELECT [fields] FROM [table]
  • SELECT [fields] FROM [table] WHERE [condition(s)]

Summary: aliasing

  • Column aliasing:
    • SELECT first_name as FirstName...
    • SELECT first_name || ' ' || last_name as FullName...
  • Table aliasing
    • SELECT u.user_name, u.uid FROM users u

Summary: filtering results

  • DISTINCT
    • SELECT DISTINCT [fields] FROM [table]
  • WHERE
    • SELECT [fields] FROM [table] WHERE [condition(s)]

Summary: WHERE clause operators

  • field = value
    • field = 'value' for strings, dates, some other types
    • field = value for numbers, booleans, some other types
    • Operators: =, >, <, >=, <=, !=
  • logical combiners
    • Both must be true: field = value AND other_field = other_value
    • Either could be true: field = value OR other_field = other_value

Summary: WHERE partial matching using LIKE

  • LIKE: Partial matches
  • Wildcard: %
    • last_name LIKE 'P%'
    • May be case insensitive, depending on the database (it is in MySQL, mostly)
  • Single character: _
    • last_name LIKE 'PA_TON'

Summary: More WHERE operators: BETWEEN and IN

  • field BETWEEN value1 AND value2:
    • effectively field >= value1 AND field <= valuu2
    • Inclusive of value1 and value2
  • field IN ('value1', 'value2', ...)
    • A set of possible values for field
    • Shorter version of multiple OR clauses

Summary: NULL

  • NULL is a special value indicating unknown data
    • Not the same as blank or zero
    • NULL is not equal to NULL, because one unknown value cannot be known to be equal to another unknown value
  • Detect NULL with IS
    • WHERE field IS NULL
    • WHERE field IS NOT NULL
